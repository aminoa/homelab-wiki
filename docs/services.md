    Simple Scratch game I wrote in 8th grade, one of my first programs

- [CTFd](https://recruit.aneeshmaganti.com)

    Capture the Flag challenges I made while I was at NYU across CSAW 2023/2024; if you can prove that beat Cell without looking at the solution online, I'll buy you a burrito bowl at Chipotle (for the first 5 non-CSAW winners).

- [DAC](https://dac.aneeshmaganti.com)
  
    A speculative-fiction interactive Unity game where you explore an abandoned Climate facility far in the future; I made this for my Climate Fiction class back in 2022. In 2026, I decided to make a WASM port which was more challenging than expected due to my incomplete source code (requiring using AssetExtractor from an existing Windows build of the game) and multiple compilation/missing script errors which required hand-fixes.
